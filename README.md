## Algorithm
### 복잡도   
N의 범위가 500인 경우 : O(n^3)인 알고리즘을 설계하면 풀 수 있다.   
N의 범위가 2,000인 경우 : O(n^2)   
N의 범위가 100,000인 경우 : O(nlogn)   
N의 범위가 10,000,000인 경우 : O(n)
